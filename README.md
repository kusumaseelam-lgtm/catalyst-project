# AI Talent Scouting System

## Overview
This project is an AI-powered talent scouting platform that analyzes candidate profiles and provides scoring based on skills and relevance.

## Features
- Resume / profile input
- AI-based scoring
- Streamlit web interface
- Simple and fast evaluation

## Tech Stack
- Python
- Streamlit

## How to Run
1. Install requirements:
pip install -r requirements.txt

2. Run app:
streamlit run app.py
